Lägg till Jobbsidan Om sidan

Om sidan

Denna sida är skapad som lösning till uppgift 3.2 i kursen "Backend-baserad webbutveckling".

Webbsidan är en plattform som gör att du kan registrera dina arbetserfarenheter till ett CV. På "Lägg till"-sidan kan du inkludera din arbetslivserfarenhet, som sedan syns på startsidan. För varje jobb kan du ange företagsnamn, titel, plats och arbetsbeskrivning.

Plattformen använder Fetch API för att hämta (GET), skicka (POST) och radera (DELETE) data till och från en extern webbtjänst (skapad i uppgift 3.1), som i sin tur är kopplad till en MongoDB-databas. Både webbtjänsten och databasen är publicerad på render.com.

Uppgiftens första del (databasen) var ganska klurig dåd et var mycket nytt. Men det kändes som att skapandet hade ett enklare system med MongoDB och Mongoose, jämfört med PostgreSQL som jag tidigare använt.
